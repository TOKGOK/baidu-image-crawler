# 项目：百度图库图片爬虫

## 🎯 项目目标

开发一个可靠、高效的百度图库图片爬虫，支持断点续传、多线程并发和持久化日志。

## 📋 核心功能

| 功能 | 优先级 | 状态 |
|------|--------|------|
| 断点续传机制 | P0 | ⏳ 待开发 |
| 默认存储路径配置 | P0 | ⏳ 待开发 |
| 操作状态记忆（持久化日志） | P0 | ⏳ 待开发 |
| 自定义线程池配置 | P1 | ⏳ 待开发 |
| 图片下载进度显示 | P1 | ⏳ 待开发 |
| 错误重试机制 | P1 | ⏳ 待开发 |
| Cookie/Token 安全处理 | P0 | ⏳ 待开发 |

## 🚫 项目范围

**包含**：
- 百度图库图片搜索和下载
- 断点续传和状态持久化
- 多线程并发下载
- 日志和状态管理

**不包含**：
- 其他图片网站爬取
- 图片处理和编辑
- 用户界面（GUI）

## 🔐 安全要求

- ❌ 严禁提交 Cookie、Token 等敏感信息
- ✅ 使用环境变量存储凭证
- ✅ .gitignore 排除敏感文件
- ✅ 提交前自动安全审计

## 📊 成功标准

- [ ] 支持断点续传，中断后可恢复
- [ ] 支持自定义线程数（默认 5 线程）
- [ ] 日志持久化，重启后保留状态
- [ ] 无敏感信息泄露
- [ ] 代码通过安全审计
