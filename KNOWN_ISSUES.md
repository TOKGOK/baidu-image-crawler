4. 优化请求头配置，添加 `Referer` 和 AJAX 标识
5. 实现双策略并行搜索（JSON API 优先，HTML 解析备用）

**当前状态**：✅ 功能已恢复正常，无需配置 Cookie 即可获取真实主题图片

## 🔧 可选优化

### 配置百度 Cookie（可选优化）⭐

虽然修复后无需 Cookie 即可正常使用，但配置 Cookie 可进一步提高成功率：

```bash
# 1. 获取百度 Cookie
#   - 打开 https://image.baidu.com/
#   - 按 F12 打开开发者工具
#   - 搜索任意关键词
#   - 在 Network 标签找到请求
#   - 复制 Request Headers 中的 Cookie

# 2. 创建 .env 文件（如果不存在）
# 3. 添加 Cookie 配置
BAIDU_COOKIE=BAIDUID=xxx; BDUSS=xxx; STOKEN=xxx; ...

# 4. 重新运行
python main.py "派蒙" 50
```

## 🔧 其他问题

### 问题 2: 下载速度慢

**原因**：
- 网络连接不稳定
- 百度服务器响应慢

**解决**：
- 检查网络连接
- 减少并发线程数（修改 MAX_THREADS）

### 问题 3: 下载中断

**解决**：
- 程序支持断点续传
- 直接重新运行即可恢复

### 问题 4: 部分图片下载失败

**原因**：
- 图片源服务器不可用
- 图片 URL 已失效

**解决**：
- 程序会自动重试（最多 3 次）
- 失败的图片会记录在日志中
